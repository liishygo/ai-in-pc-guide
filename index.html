為什麼要在電腦端用 AI？

你可能已經用過 ChatGPT 或 Gemini 的網頁版。但在電腦端使用 AI，本質上是完全不同的體驗——

比較項目 瀏覽器 AI（ChatGPT、Gemini） 電腦端 AI（Antigravity、Claude Code、Windsurf）
本質 一問一答的對話 AI 可以在你的電腦裡執行任務
檔案存取 手動上傳 / 下載 直接讀寫你電腦裡的檔案
工作上下文 只看得到你貼進對話的內容 自動看見整個資料夾：所有檔案、架構、彼此關聯
工作模式 以對話為核心，每個專案只有一組預設指令，主要優化對話的輸出品質 以工作流程為核心，制訂規則、固定流程、技能、工具等等，涵蓋對話、計畫、執行、初步成果、修改、最終結果等多個階段
產出方式 給你文字，你自己複製貼上 直接幫你建檔、改檔、存檔
資料歸屬 檔案上傳到雲端才能使用 資料不離開你的電腦，檔案很多時，會很方便

軟體安裝

從下載到登入，5 分鐘完成
🍎

macOS

macOS Monterey 12 以上
支援 Apple Silicon 及 Intel

🪟

Windows

Windows 10（64-bit）以上
支援 x64 及 ARM64

1

前往官網下載

開啟瀏覽器，前往 antigravity.google/download
依你的電腦系統選擇版本：Apple Silicon / Intel / Windows x64

2

執行安裝程式

macOS：將下載的 .dmg 拖曳到 Applications 資料夾即完成安裝
Windows：雙擊 .exe 安裝程式，依畫面指示完成安裝步驟

3

登入 Google 帳號

首次啟動時選擇「Sign in with Google」，使用公司或個人 Google 帳號授權即可。無需額外付費。

✅ 安裝完成後，畫面會出現三欄式工作介面，代表成功啟動！

AI 最愛的檔案格式 Markdown

用 AI 的人，一定要認識這個格式
Markdown 是 AI 最愛讀的格式——純文字、檔案小，還能設定標題、列點、重點標示。最重要的是：餵 Markdown 給 AI，可以大幅節省 token 消耗。

✍️ 寫起來像這樣

# 文章標題

## 第一段落
這是內文，可以 **粗體** 或 *斜體*。

- 重點一
- 重點二
- 重點三

💡 為什麼要用 Markdown？

  • AI 可以直接理解結構，不需要解析排版
  • 比 Word / PDF 消耗更少 token
  • 純文字，任何裝置都能開啟
  • Antigravity 原生支援，有即時預覽
📌 你有 Word、PDF 要給 AI 分析？先轉成 Markdown，再貼給 AI，效果更好、花費更少。

延伸模組安裝

精選 4 個內容創作者入門必裝模組

Antigravity 與 VS Code 共用相同的延伸模組生態，你可以安裝數千種工具來強化編輯體驗。以下是針對內容創作者精選的 4 個入門必裝模組。

1

開啟延伸模組面板

點擊左側欄的「Extensions」圖示（方塊圖案），或使用快捷鍵 ⌘⇧X（Mac）/ Ctrl+Shift+X（Windows）

2

更改 Marketplace 網址

按一下「Antigravity settings」，將以下兩個欄位填入對應網址：

Marketplace Item URL
https://marketplace.visualstudio.com/items
Marketplace Gallery URL
https://marketplace.visualstudio.com/_apis/public/gallery
3

搜尋並安裝

在搜尋框輸入下方的模組名稱，找到對應項目後點擊「Install」即開始安裝

4

重新載入生效

安裝完成後，若出現「Reload Required」提示，點擊後即可正常使用

基本設定：Rules

給 AI 的長期身份卡，設定一次永久生效
Rules 是你給 AI 的長期身份卡——告訴它你是誰、你的文章風格、輸出格式要求。設定一次，每次對話都會自動套用，不需要每次重新說明。
右側 AI 對話區
→
點右上角 齒輪圖示
→
選「Rules」
→
貼入你的規則後儲存

範例 Rules（可直接複製使用）：

# 我的身份
我是城邦文化的內容編輯，主要負責科技與生活類文章。

# 語氣風格
- 親切、白話，避免技術術語
- 段落簡短，每段不超過 4 句
- 適合 25-40 歲讀者閱讀

# 輸出格式
- 語言：繁體中文
- 段落分明，重點可用條列式
- 標題使用 ## 語法
- 圖片說明文字：每張圖 1 句，不超過 20 字
💡 小技巧：Rules 不需要完美，先從「語言：繁體中文」開始，之後邊用邊補充即可。
⚙️

基本設定：Workflows

把常用多步驟指令存成一鍵範本
Workflows 是把你最常用的多步驟指令，存成一個一鍵啟動的範本。免得每次都要重新打一長串指令，輸入觸發詞即可執行完整流程。
📌 Rules

永遠有效

  • 一次設定，每次對話自動套用
  • 適合：身份、語氣、格式等固定規則
  • 不需要主動啟動，背景持續運作
▶️ Workflows

需要時才啟動

  • 輸入觸發詞，執行多步驟任務
  • 適合：產稿、改稿、格式轉換等流程
  • 可設定多個 Workflows 分工使用

Skills 進階設定

擴充 AI 能力，讓它學會特定領域專業技能
Skills 是比 Workflows 更進一步的能力擴充包，可以讓 AI 學會特定領域的專業技能，例如 SEO 優化、圖片描述生成、資料結構化等。
項目 Workflows Skills
設定難度 ⭐ 簡單 ⭐⭐ 中等
適合對象 所有人 進階使用者
功能範圍 固定流程範本 擴充 AI 能力
安裝方式 設定介面填寫 安裝 Skill 檔案
1

開啟 Skills 面板

在右側 AI 對話區，點擊上方「Skills」圖示，進入 Skills 管理介面

2

瀏覽官方目錄

瀏覽或搜尋官方 Skills 目錄，依你的工作需求找到對應的技能包

3

安裝並確認權限

點「Add Skill」，確認所需存取權限後完成安裝，重新載入即可使用

💡 建議先熟悉 Rules 和 Workflows 後，再來探索 Skills。大部分內容工作者靠前兩項就足夠了！
