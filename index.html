為什麼要在電腦端用 AI？

你可能已經用過 ChatGPT 或 Gemini 的網頁版。但在電腦端使用 AI，本質上是完全不同的體驗——

比較項目 瀏覽器 AI：ChatGPT、Gemini 電腦端 AI：Antigravity(Gemini)、Claude CoWork、Codex(ChatGPT)
本質一問一答的對話AI 可以在你的電腦裡執行任務，多個AI可互相合作
檔案存取手動上傳 / 下載、複製 / 貼上直接讀寫你電腦裡的檔案
工作上下文只看得到你貼進對話的內容或上傳的檔案自動看見整個資料夾：所有檔案、架構、彼此關聯
工作模式以「對話」為主，提供答案後，由使用者自己整理或執行以「做事」為主，運用規則、流程、技能、工具等等，進行對話、計畫、執行、修改、交付成果等等多個階段
產出方式給你文字，你自己複製貼上直接幫你建檔、改檔、存檔
資料歸屬檔案上傳到雲端才能使用資料不離開你的電腦，檔案很多時，會很方便
功能擴展只能用廠商給你的功能，想要的功能慢慢等有新的需求、需要新的功能，叫 AI 從研究、選擇、安裝到使用自行完成

AI 最愛的檔案格式 Markdown

用 AI 的人，一定要認識這個格式

Markdown 是 AI 最愛讀的格式——純文字、檔案小，還能設定標題、列點、重點標示。最重要的是：餵 Markdown 給 AI，可以大幅節省 token 消耗。

✍️ 寫起來像這樣

# 文章標題

## 第一段落
這是內文，可以 **粗體** 或 *斜體*。

- 重點一
- 重點二
- 重點三

💡 為什麼要用 Markdown？

  • AI 可以直接理解結構，不需要解析排版
  • 比 Word / PDF 消耗更少 token
  • 純文字，任何裝置都能開啟
  • Antigravity 原生支援，有即時預覽
📌 你有 Word、PDF 要給 AI 分析？先轉成 Markdown，轉檔工作交給 AI 就對了。

安裝 Antigravity

從下載到登入，5 分鐘完成

🍎

macOS

macOS Monterey 12 以上
支援 Apple Silicon 及 Intel

🪟

Windows

Windows 10（64-bit）以上
支援 x64 及 ARM64

1

前往官網下載

開啟瀏覽器，前往 antigravity.google/download
依你的電腦系統選擇版本：Apple Silicon / Intel / Windows x64

2

執行安裝程式

macOS：將下載的 .dmg 拖曳到 Applications 資料夾即完成安裝
Windows：雙擊 .exe 安裝程式，依畫面指示完成安裝步驟

3

登入 Google 帳號

首次啟動時選擇「Sign in with Google」，使用公司或個人 Google 帳號授權即可。無需額外付費。

Antigravity延伸模組

精選 4 個內容創作者入門必裝模組

Antigravity 與 VS Code 共用相同的延伸模組生態，你可以安裝數千種工具來強化編輯體驗。以下是針對內容創作者精選的 4 個入門必裝模組。

1

開啟延伸模組面板

點擊左側欄的「Extensions」圖示（方塊圖案），或使用快捷鍵 ⌘⇧X（Mac）/ Ctrl+Shift+X（Windows）

2

更改 Marketplace 網址

按一下「Antigravity settings」，將以下兩個欄位填入對應網址：

Marketplace Item URL
https://marketplace.visualstudio.com/items
Marketplace Gallery URL
https://marketplace.visualstudio.com/_apis/public/gallery
3

搜尋並安裝

在搜尋框輸入下方的模組名稱，找到對應項目後點擊「Install」即開始安裝

4

重新載入生效

安裝完成後，若出現「Reload Required」提示，點擊後即可正常使用

安裝 Codex

讓 ChatGPT 在電腦裡面做事

🍎

macOS

macOS Ventura 13 以上
支援 Apple Silicon 及 Intel

🪟

Windows

Windows 10（64-bit）以上
支援 x64 及 ARM64

1

前往官網下載

開啟瀏覽器，前往 openai.com/zh-Hant/codex
頁面會自動偵測你的作業系統，顯示對應的下載按鈕，點擊即可下載安裝程式。

2

macOS：開啟下載的 .dmg，將 Codex 拖曳到 Applications 資料夾完成安裝

3

登入 ChatGPT 帳號

首次啟動時選擇「使用 ChatGPT 帳號登入」，使用你的 ChatGPT 帳號登入即可。

✅ 登入後看到對話介面，就可以開始用自然語言指派任務，Codex 會自動操作你電腦上的檔案與資料夾。

左下角 Settings
→
選單中按 Settings
→
左側選單按 General
→
找到 Language，改為「中文(台灣)」
